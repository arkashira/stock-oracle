# Stock Oracle

A simple stock management system.

## Usage

1. Create a `StockOracle` instance.
2. Add stock items using `add_stock_item`.
3. Load data from a list of dictionaries using `load_data`.
4. Get the dashboard data using `get_dashboard_data`.
5. Get the reorder points using `get_reorder_points`.

## Testing

Run the tests using `pytest`.
