# ROADMAP.md

This roadmap outlines the development plan for the `stock-oracle` project, focusing on delivering a robust inventory replenishment planning system with incremental enhancements.

## MVP (Minimum Viable Product)
The MVP will provide core functionality to manage inventory levels and automate reorder processes, addressing the immediate needs of users.

### MVP-Critical Features
- **Product Catalog Management**: CRUD operations for products (add, edit, delete).
- **Inventory Level Tracking**: Real-time tracking of stock levels across locations.
- **Reorder Point Calculation**: Automated calculation of reorder points based on lead time and safety stock.
- **Low Stock Alerts**: Notifications when stock falls below the reorder point.
- **Basic Inventory Reports**: Summarized reports on stock levels and reorder history.

## Phase 1: Foundation & Core Features
Target: 3 months post-MVP release. Focus on scalability and user management.

### Theme: Integration & Scalability
- **MVP-Critical (continued)**:
  - Supplier Integration: API integration with external supplier systems for real-time catalog updates.
  - User Roles & Permissions: Role-based access control (admin, regular user).
  - Detailed Reporting: Exportable reports (CSV/Excel) for stock levels and reorder trends.
- **New Features**:
  - Batch Operations: Support for bulk adding/editing products.
  - Inventory Audit Logs: Track changes to inventory levels and reorder points.
  - Multi-Location Support: Extend functionality to manage inventory across multiple warehouses.

## Phase 2: Advanced Analytics & Automation
Target: 6 months post-MVP release. Introduce predictive analytics and workflow automation.

### Theme: Predictive Analytics & Workflow Automation
- **New Features**:
  - Demand Forecasting: Machine learning-based demand prediction using historical sales data.
  - Automated Purchase Orders: Generate purchase orders based on forecasts and reorder points.
  - Accounting Integration: Sync with accounting systems to track costs and revenue.
  - Advanced Analytics: Forecast vs. actual comparison and cost-benefit analysis.

## Phase 3: Enterprise Features (Future)
- **Theme: Enterprise-Grade Capabilities**
  - Advanced Role-Based Access: Fine-grained permissions for different departments.
  - Integration with ERP Systems: Full ERP system integration.
  - Customizable Workflows: Define custom reorder rules and alerts.
  - Mobile App: Mobile access for on-the-go inventory management.

This roadmap ensures a phased approach, starting with core functionality and progressively adding value through integration, automation, and advanced analytics.
